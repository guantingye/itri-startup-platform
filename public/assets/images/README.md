# 📁 產品圖片上傳位置

## 規格要求

- **尺寸**: 800x450px (16:9比例)
- **格式**: JPG / PNG
- **大小**: < 500KB
- **命名**: 使用小寫英文和連字符

## 檔案清單

請將以下產品示例圖片放置在此目錄：

1. `asml-euv-machine.jpg` - ASML EUV光刻機圖片
2. `asml-wafer.jpg` - ASML晶圓圖片
3. `bosch-bms.jpg` - Bosch BMS系統圖片

## 使用方式

上傳後重新啟動應用：

```bash
npm start
```

圖片會自動在訂單合作頁面的產品需求區域顯示。

## 注意事項

- 如果沒有上傳圖片，會顯示占位符
- 確保檔名與程式碼中的路徑一致
- 建議使用JPG格式以減小檔案大小
