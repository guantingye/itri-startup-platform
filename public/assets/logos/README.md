# 📁 Logo上傳位置

## 規格要求

- **尺寸**: 120x120px (正方形)
- **格式**: PNG (建議透明背景)
- **大小**: < 100KB
- **命名**: 使用小寫英文和連字符

## 檔案清單

請將以下公司Logo放置在此目錄：

1. `asml-logo.png` - ASML公司Logo
2. `mediatek-logo.png` - 聯發科Logo  
3. `bosch-logo.png` - Bosch公司Logo

## 使用方式

上傳後重新啟動應用：

```bash
npm start
```

Logo會自動在訂單合作頁面的公司簡介區域顯示。
